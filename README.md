# hopechild-drc-management
Application de gestion des données de HOPE CHILD DRC permettant de centraliser, organiser et sécuriser les informations relatives au personnel, matériels, projets, bénéficiaires, enquêteurs et partenaires.  Fonctionne en mode hors ligne (offline) avec tableau de bord, gestion des rôles, recherche avancée et export des données.
